Tagclouds

Module to generate TagCloud images based on music databases

tagclouds.frequency(text)

Receives a string, and returns a multidict with each word’s number of ocurrencies in the string

Parameters

text (str) – A string with the text wished to count each word’s frequency

Returns

A Multidict with words as keys and the word’s frequencies as values

Return type

multidict.MultiDict

tagclouds.frequency_generator(frequencies, file_name)

Receives a frequency MultiDict, and creates a tagcloud named ‘{file_name}.png’

Parameters

  • frequencies (multidict.MultiDict) – MultiDict with words as keys and frequencies as values

  • file_name (str) – string that defines the name of the image file to be saved

tagclouds.generate_cloud_albuns(df)

Receives a DataFrame with musics and generates a TagCloud image named ‘{album_name}_tagcloud.png’ according to the most frequent words in each album’s lyrics

Parameters

df (pd.DataFrame) – A DataFrame with an index called ‘Album’ and a ‘Lyrics’ column, with all music’s lyrics as strings

tagclouds.generate_cloud_lyrics(df)

Receives a DataFrame with musics and generates a TagCloud image named ‘all_lyrics_tagcloud.png’ according to the most frequent words in all lyrics

Parameters

df (pd.DataFrame) – A DataFrame with a ‘Lyrics’ column, with all music’s lyrics as strings

tagclouds.generate_cloud_music_names(df)

Receives a DataFrame with musics and generates a TagCloud image named ‘names_tagcloud.png’ according to the most frequent words in all music’s titles

Parameters

df (pd.DataFrame) – A DataFrame with an index called ‘Name’, with all music’s names as strings

tagclouds.lyrics_albuns(df)

Receives a dataframe with musics and return a dictionary with all albums names and all music’s lyrics in each album concatenated as a string.

Parameters

df (pd.DataFrame) – Dataframe with an ‘Album’ Multi-Index (the name of the album each music belongs to), and a ‘Lyrics’ column with each song’s lyrics as strings

Returns

Dictionary with albums names as keys, and all music’s lyrics in each album concatenated as a string, as the key’s values.

Return type

dict

tagclouds.lyrics_all(df)

Receives a DataFrame with musics and returns a string with every lyric concatenated

Parameters

df (pd.DataFrame) – musics DataFrame with one column ‘Lyrics’ that has each music’s lyrics as strings

Returns

string with every artist’s music lyrics concatenated

Return type

str

tagclouds.music_names(df)

Receives a dataframe with musics and returns a concatenated string with every song’s names

Parameters

df (pd.DataFrame) – Dataframe with a Multi-Index called ‘Music’ (music’s name)

Returns

string with all musics names concatenated

Return type

str
