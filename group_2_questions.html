Group 2 Questions

Module that contains the functions to answer the questions of the second group of questions.

group_2.dict_to_series(dict)

Receives a dictionary with words and frequency numbers and returns a Series object sorted by descending values

Parameters

dict (dict | multidict.MultiDict) – Dictionary with words as keys and numbers as values

Returns

Pandas Series with words as keys and numbers as values, sorted by descending values

Return type

pd.Series
