SpotifyAPI

Module that takes The Neighbourhood songs data from the Spotify API.

spotify_api.get_albums_data()

Function that makes the request to get all the albums ids and names of an artist.

Returns

Tuple with the names and ids.

Return type

tuple

spotify_api.get_albums_tracks()

Function that makes the request to get all the tracks ids of an album.

Returns

Tracks id and in wich album it is in.

Return type

dict

spotify_api.get_tracks_data()

Gets all the information we need about the tracks,

Returns

The name, popularity, duration, explicit, track number and available markets of al tracks of an artist.

Return type

dict

Make the request using and spotify object and the artist id.
