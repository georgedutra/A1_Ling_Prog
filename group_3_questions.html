Group 3 Questions

Module that contains the functions to answer the questions of the third group of questions.

group_3.question_1(df)

Receives a DataFrame with musics information, and print at the terminal wich album has the higher music duration mean.

Parameters

df (pd.DataFrame) – DataFrame with an index named ‘Album’ and a column named ‘tracks_duration_ms’ with the track’s duration as numbers (in miliseconds)

group_3.question_2(df)

Receives a DataFrame with musics information, and print at the terminal wheter the music’s popularity is related to the lyrics being explicit or not

Parameters

df (pd.DataFrame) – DataFrame with a column named ‘tracks_popularity’ with numbers and other column named ‘tracks_explicit’ with booleans

group_3.question_3(df)

Receives a DataFrame with musics information, and print at the terminal wich album has the higher popularity mean.

Parameters

df (pd.DataFrame) – DataFrame with an index named ‘Album’ and a column named ‘tracks_popularity’ with the track’s popularity as a number.
