A1 - Programming Language: The Neighbourhood data Analysis

Project developed for the Programming Languages discipline at EMAP FGV. The current project makes an scrapping at Letras and Spotify for Developers to collect data of The Neighbourhood. Using this data, we make some analysis and graphics to respond the proposed questions.
