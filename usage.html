Usage

Installation

First install the used libraries using pip:

$ pip install requirements.txt

Scrapping Data

To retrieve a dictionary with all the artist songs from SpotifyAPI, you can use the spotify_api.get_tracks_data() function:

spotify_api.get_tracks_data()

Gets all the information we need about the tracks,

Returns

The name, popularity, duration, explicit, track number and available markets of al tracks of an artist.

Return type

dict

To create a dictionary with all the lyrics from Letras, you can use the mus_br_letras.get_nbhd_songs() function:

mus_br_letras.get_nbhd_songs()

Function that takes The Neighbourhood songs data

Returns

The Neighbourhood songs data, the key is the song name.

Return type

dict

Finally, if you want to generate a .csv so you don’t have to do the scrapper all the time, with all the information, you can use create_df.create_final_dataframe() function:

create_df.create_final_dataframe()

Function that aggregates the necessary functions and generates the .csv and returns the dataframe in case you need it.

Parameters

csv_name – Name of the .csv that will be generated. Must have the “.csv at the end.

Returns

Final dataframe with lyrics in it.

Return type

pandas.core.frame.DataFrame
